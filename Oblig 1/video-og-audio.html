video og audio fungerer i HTML5

Det er viktig å skille mellom innhold og design siden det skaper en del fordeler når man utvikler en nettside. Hvis vi blander mellom HTML og CSS kan det bli mye rot i selve nettsiden, og det kan bli vanskelig for leseren som leser nettsiden til å tolke informasjonen som er på nettsiden. Siden HTML er skjelettet i nettsiden som består av head og body. Ved hjelp av HTML kan man formidle alle den informasjonen man vil ha på nettsiden. CSS er selve utsiden det er presentasjonen av skjelettet som er HTML. For at vi ikke skal gjøre nettsiden kaotisk og uorganisert så ha vi en tommelregel. Den går ut at vi skal bruke CSS til alt som ikke er informasjon. DVs at ting som bakgrunnsfarge, tekst størrelse og bilde plassering osv. den HTML-versjonen som vi bruker i dag som er HTML5 er et såkalt markeringsspråk. DVs at man markerer et ordinert dokument med tagger for å teksten spesiell betydning. CSS som sagt så går CSS ut på hvordan ut sene er. Vi kan tenke på CSS som design. HTML står for (Hyper Text Markup Language). CSS står for (Cascading Style Sheets).
