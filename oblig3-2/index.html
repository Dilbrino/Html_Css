Html skjema
Photo by Maxwell Nelson on Unsplash

Et webskjema, webskjema eller HTML-skjema på en webside tillater en bruker å legge inn data som sendes til en server for behandling. Skjemaer kan ligne på papir- eller databaseskjemaer fordi nettbrukere fyller ut skjemaene ved hjelp av avkrysningsruter.

Css layouts
Photo by Sigmund on Unsplash

De fleste websider er skrevet i en kombinasjon av HTML (Hypertext Markup Language) og CSS (Cascading Style Sheets). Kort sagt spesifiserer HTML innholdet og den logiske strukturen på siden, mens CSS spesifiserer hvordan den ser ut: dens farger, skrifttyper, formatering, layout og styling
